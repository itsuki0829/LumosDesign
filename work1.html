WORKS

- WORKERS -
会員制コワーキングスペースのブランドサイト

WEBサイト スクール課題

概要
会員制コワーキングスペース「WORKERS」を想定したブランドサイトを制作。
「働く人が安心して過ごせる場所」をコンセプトに、
信頼感と落ち着きを感じられるデザインを目指しました。
初めて訪れるユーザーにも施設の雰囲気や利用イメージが伝わるよう、
情報の整理と視認性を意識しています。
目的
コワーキングスペースの特徴や魅力を分かりやすく伝え、
利用を検討しているユーザーに安心感を与えること。
見学や入会へのアクションにつなげることを目的としています。
ターゲット
フリーランス・リモートワーカー・個人事業主など、
落ち着いた環境で作業できる場所を探している20〜40代の働く人。
制作期間
約1週間
使用ツール
Adobe Illustrator / Photoshop / Visual Studio Code
